### Game Setup
First, download this zip file: https://github.com/Lunarienya/Minecraft/raw/master/Lunarie.zip  
**DO NOT EXTRACT IT.**

To start playing Minecraft, you need to add an "instance."
Click on the "Add Instance" button inside Prism Launcher, then go to the
"Import" option.

![](https://safe.saya.moe/Ukllwwt6z91p.png)

Click the Browse button and select the zip file that was downloaded.
Now you can play! Just click the Launch button, or double click
the instance (with the cat icon).

The server will already be in the multiplayer server list.
