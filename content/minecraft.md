Lunarie hosts a Minecraft server for all (current and former) Lunarie
service users. These are the steps on how you can connect.

# Acquiring Minecraft
The Lunarie Mineraft server is a "cracked" server to allow anyone
with an account to join. Here is the way you can get Mineraft to join.

## TLauncher/Legacy Launcher
TLauncher can be downloaded from [this website](https://llaun.ch/en).
Legacy is the real TLauncher, do NOT download the launcher named
TLauncher.

Note that no support will be provided for using Legacy Launcher.
Only use it if you know how to get the required mods to join.

## PrismLauncher
Prism is the recommend laucnher and personal pick of administrators of Lunarie.
Download and install it from [this link.](https://github.com/Diegiwg/PrismLauncher-Cracked/releases/download/7.1/PrismLauncher-Windows-MSVC-Setup-7.1.exe)

### Account Setup
Add your account:
![](https://safe.saya.moe/AXHE1Ag8drv6.png)

1. Click on "Manage Accounts"
2. Then on the new window that opens, click "Add Offline" and enter
the username you want.

### Game Setup
First, download this zip file: https://safe.saya.moe/JGMgSDqcHXJp.zip
You can rename it to anything you want.

To start playing Minecraft, you need to add an "instance."
Click on the "Add Instance" button inside Prism Launcher, then go to the
"Import" option.

Click the Browse button and select the zip file that was downloaded.
Now you can play! Just click the Launch button, or double click
the instance (with the cat icon).

The server will already be in the multiplayer server list.
